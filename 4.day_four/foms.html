forms

form will have a action attribute which is used to define what action need to peformed when a form is submitted or where the form data should be sent

example :

<form action='url'>

form content

</form>

Form elements

input :

used to create multiplel form controls

there are multiple types of inputs that can be created using type attribute

it has a attribute that is placeholder used to give hints about the type of input

label :

Label element represent a caption for an item in a user interface

label classic approuch

<label for='username'> Enter your username : </label> <input type='text' id='username' placeholder='username'>

example

button :

button element used to create a button

type attributes

<button type='submit'>Submit</button>

<button type='reset'>Reset</button>

<button type='button'>Button</button>

example

Name attribute :

Name of the form control submitted with the form as a part of a name/value pair

<input type='text' name='username' id='username' placeholder='username'>

example

checkbox :

used to create a checkbox

<input type='checkbox' name='age' id='age'> <label for='age' >i am 18+ </label>

example

radio :

used to create a radio

<input type='radio' name='fruit' id='apple' value='apple'> <label for='apple' >Apple </label> <input type='radio' name='fruit' id='mango' value='mango'> <label for='mango' >mango </label>

example

dropdown

used to create a dropdown

<select name='city' id='city'> <option value='Delhi'>Delhi</option> <option value='Mumbai'>Mumbai</option> <option value='Chennai'>Chennai</option> </select>

example

range

used to create a range

min and max attribute is used to set the range minimum and maximum

here we use step attribute to change the range with a step of 10

<input type='range' name='range' id='range' min='0' max='100' step='10'>

example

textarea

used to create a text area

<label for='comment'>Enter comment here</label> <textarea name='comment' id='comment' cols='30' rows='10'>Enter comment here</textarea>
