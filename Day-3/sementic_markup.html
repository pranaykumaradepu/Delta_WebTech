Semantic Tags in HTML

What are Semantic Tags?

Semantic tags are HTML elements that provide meaning and structure to the content on a web page. They describe the purpose and role of the content, making it easier for browsers, search engines, and assistive technologies to understand and interpret the information.

Examples of Semantic Tags

  • <header> - Represents the header section of a document or a section.
  • <nav> - Defines a section for navigation links.
  • <main> - Represents the main content area of a document.
  • <article> - Represents a self-contained composition, such as a blog post or news article.
  • <section> - Defines a section in a document, typically with a heading.
  • <aside> - Represents content that is only indirectly related to the document's main content.
  • <footer> - Represents the footer section of a document or a section.
  • <details> - Represents additional details that are not displayed by default.
  • <summary> - Defines a summary or caption for the details element.
  • <time> - Represents a specific point in time.
  • <mark> - Represents text that has been highlighted for reference or explanation purposes.
